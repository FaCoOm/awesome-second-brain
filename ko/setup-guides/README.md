# 설정 가이드

설정 가이드는 세컨드 브레인 workflow를 실제로 실행하기 위한 이 레포의 직접 검증 노트입니다. 공식 제품 문서를 대체하면 안 됩니다.

설정 가이드는 아래를 포함할 때만 추가하세요.

- 목표와 대상 사용자
- 테스트한 정확한 솔루션과 client
- 필요한 prerequisites와 accounts
- 처음 쓸 만한 memory까지 가는 가장 짧은 검증된 경로
- retrieval 또는 write-back이 실제로 동작하는지 검증하는 방법
- 알려진 failure mode 또는 설정 트레이드오프

이 레포가 특정 설정 경로를 직접 검증한 경우가 아니라면 공식 설치 세부사항은 링크로 유지하세요.
