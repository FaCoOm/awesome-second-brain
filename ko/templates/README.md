# Templates

솔루션 프로필과 기능 축 정의 문서를 추가하거나 갱신할 때 이 template을 사용하세요.

| Template | 사용처 |
|---|---|
| [system-profile.md](system-profile.md) | `solutions/` 아래 솔루션 프로필. |
| [capability-page.md](capability-page.md) | `capabilities/` 아래 기능 축 정의. |

## 작업 흐름

1. 관련 template을 `solutions/` 또는 `capabilities/`로 복사합니다.
2. placeholder를 출처가 뒷받침하는 주장으로 바꿉니다.
3. 근거가 없거나 검증하지 않은 claim은 `미확인`으로 둡니다.
4. 새 문서를 README, 폴더 README, 비교 문서에서 링크합니다.
