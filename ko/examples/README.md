# Examples

Examples는 도구, 정보원, AI assistant, memory layer를 조합한 구체적인 세컨드 브레인 workflow를 보여줍니다.

아래와 같은 시나리오 수준 의사결정에 사용하세요.

- 한 사람이 여러 AI 도구를 사용하는 경우
- 팀이 project knowledge를 공유하는 경우
- note와 file 위에 local-first personal memory를 만드는 경우
- AI product 안에 developer memory API를 넣는 경우
- bounded source research와 living memory를 비교하는 경우

각 example은 관련 솔루션 프로필, 기능 축 정의, 비교 문서로 다시 연결되어야 합니다.
